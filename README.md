# 🧭 StarWarsRP — навигатор по картам

Веб-приложение для ролевых игр по Star Wars: кликаешь по карте — получаешь **координаты клетки**, **расстояние** и **азимут** между точками, а для клона/техники — ещё и **время в пути**. Работает прямо в браузере, ничего устанавливать не нужно, интернет нужен только чтобы открыть страницу (дальше — офлайн).

## 🗺️ Возможности

- **Карта**: выпадающий список из 22 карт (Phase-2) — МЦБ-17/9/3-18, Татуин, Татуин-город, Альдераан, Анаксес, Фелуция, Джеонозис (улитка/база/завод/гл. улей), Корусант-подземный, Луна Риши, Мигетто, Мигетто-2, Морабанд, Набу (ангар/лес), Рен Вар, Рен Вар-горы, Рилот, Такодана, Утапау.
- **Фаза**: кнопки Phase-1 / Phase-2 над картой. Сейчас все карты — Phase-2, поэтому при нажатии Phase-1 список карт будет пустым («— карт пока нет —») — это ожидаемо, карты для первой фазы ещё не добавлены.
- **Тип** — переключатель над картой, три режима работы:
  1. **🚶 Клон** — маршрут из любого числа точек подряд. Режимы: *Пешком*, *Лёгкий бег*, *Быстрый бег*. Каждый клик по карте добавляет новую точку маршрута. Внизу — таблица участков (расстояние + время + азимут каждого отрезка) и итоговая строка по всему маршруту.
  2. **🚁 Транспорт** — тот же маршрут из точек, режимы: *LAAT/i*, *ARC-170*, *TX-130*, *AT-RT*. Дополнительно выбирается **скорость** (100 / 300 / 1000) — время пересчитывается математически под неё.
  3. **💣 Оружие** — 2 точки «Я → Цель», только расстояние и азимут (без времени). 3-й клик сбрасывает пару и ставит новую первую точку.
- **Калибровка времени** — под переключателем режима есть поле «Калибровка» (секунд на клетку), редактируется прямо в интерфейсе без правки кода — как и поле «Сторона клетки». Пока для режима не введено число, время показывается как «—».

  Текущее состояние:

  | Тип | Режим | Калибровано? | Значение |
  |---|---|---|---|
  | Клон | Пешком | ✅ | 100 с/клетку |
  | Клон | Лёгкий бег | ✅ | 35 с/клетку |
  | Клон | Быстрый бег | ✅ | 22 с/клетку |
  | Транспорт | LAAT/i | ✅ | 18.53 с/клетку при скорости 100 |
  | Транспорт | ARC-170 | ❌ нет | — |
  | Транспорт | TX-130 | ❌ нет | — |
  | Транспорт | AT-RT | ❌ нет | — |

  Значения для ARC-170, TX-130 и AT-RT ещё предстоит замерить — как только они появятся, их можно будет вписать прямо в поле «Калибровка» в интерфейсе (значение сбросится при перезагрузке страницы) или сразу в код (см. «Обновление» ниже).

- **1-й клик** — точка маршрута/моя позиция; для «Клон»/«Транспорт» клики добавляют точки подряд, для «Оружие» 3-й клик — новая пара точек. Точки можно перетаскивать мышью/пальцем.
- Показывает: координаты клеток (напр. `H8`), расстояние (в единицах и клетках), азимут (0–360°) и линию между точками.
- **Сторона клетки** задаётся вручную (по умолчанию 135 ед.), меняется под конкретную карту при необходимости.
- **Направление севера** переключается кнопкой «🧭 Север» (поворот на 90° за клик).
- **Зум**: колесо мыши / кнопки `+ − ⟲` / щипок на телефоне. **Панорама**: правая кнопка мыши, средняя кнопка или Ctrl+перетаскивание.
- **«↶ Убрать последнюю»** — удаляет последнюю точку маршрута. **«✕ Сброс»** — очищает всё.

Азимут: `0° = N`, по часовой стрелке (E = 90°, S = 180°, W = 270°) — относительно выбранного направления севера.

## 💻 Запуск локально

Нужны `index.html` и все файлы карт (`*.jpg`, `*.png`) из репозитория — все вместе, в одной папке.

### Вариант 1 — через Git

```bash
git clone https://github.com/Lereactor/starwars-navigator.git
cd starwars-navigator
```

Затем открой **`index.html`** двойным кликом в браузере.

### Вариант 2 — скачать архивом (без Git)

1. Открой https://github.com/Lereactor/starwars-navigator
2. Нажми зелёную кнопку **Code** → **Download ZIP**.
3. **Распакуй** архив (важно — не запускать прямо из архива, иначе картинки не подтянутся).
4. Открой **`index.html`** двойным кликом.

> Все файлы должны лежать вместе в одной папке — `index.html` грузит карты по относительным путям. Интернет для работы не нужен, приложение полностью офлайн.

## 🔄 Обновление / добавление карт и режимов

- Карты и их параметры (размер сетки, направление строк, сторона клетки, фаза) описаны в массиве `MAPS` внутри `index.html`. Чтобы добавить карту — положи изображение рядом и добавь строку в `MAPS` (в т.ч. можно добавить карты с `phase: 1`, тогда кнопка «Phase-1» перестанет быть пустой).
- Типы расчёта и режимы (клон/транспорт/оружие) описаны в объекте `CATEGORIES` внутри `index.html`. Калибровочные значения (`secPerCell` / `secPerCell100`) заданы там по умолчанию — их можно поменять прямо в открытом приложении полем «Калибровка» (без правки кода), но при перезагрузке страницы вернутся значения по умолчанию из кода, так что итоговые замеры лучше сразу вписывать и в код.
