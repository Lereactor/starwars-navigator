# 🧭 StarWars Phase 2 — навигатор по картам

Веб-приложение для расчёта **расстояния** и **азимута** между двумя точками на игровых картах.
Работает в браузере, ничего устанавливать не нужно.

## 🌐 Открыть онлайн (проще всего)

**https://lereactor.github.io/starwars-navigator/**

Просто перейди по ссылке — приложение откроется в браузере. Можно скинуть друзьям эту ссылку, больше ничего не требуется.

Репозиторий: https://github.com/Lereactor/starwars-navigator

## 🗺️ Возможности

- Карты: **МЦБ** (15×15), **Татуин** (14×14), **Татуин-город** (14×14) — переключаются в списке «Карта».
- **1-й клик** — моя позиция, **2-й клик** — цель, **3-й клик** — новая позиция. Точки можно перетаскивать.
- Показывает: координаты клеток (напр. `H8`), расстояние (в единицах и клетках), азимут (0–360°) и линию между точками.
- **Сторона клетки** задаётся вручную (по умолчанию 135 ед.).
- **Направление севера** переключается кнопкой «🧭 Север» (поворот на 90°).
- **Зум**: колесо мыши / кнопки `+ − ⟲` / щипок на телефоне. **Панорама**: правая кнопка мыши или Ctrl+перетаскивание.

Азимут: `0° = N`, по часовой стрелке (E = 90°, S = 180°, W = 270°) — относительно выбранного направления севера.

## 💻 Запуск локально

Нужны 4 файла в одной папке: `index.html`, `field.jpg`, `tatooine.png`, `tatooine-city.png`.

### Вариант 1 — через Git

```bash
git clone https://github.com/Lereactor/starwars-navigator.git
cd starwars-navigator
```

Затем открой **`index.html`** двойным кликом в браузере.

### Вариант 2 — скачать архивом (без Git)

1. Открой https://github.com/Lereactor/starwars-navigator
2. Нажми зелёную кнопку **Code** → **Download ZIP**.
3. **Распакуй** архив (важно — не запускать прямо из архива, иначе картинки не подтянутся).
4. Открой **`index.html`** двойным кликом.

> Все файлы должны лежать вместе в одной папке — `index.html` грузит карты по относительным путям. Интернет для работы не нужен, приложение полностью офлайн.

## 🔄 Обновление / добавление карт

Карты и их параметры (размер сетки, направление строк, сторона клетки) описаны в массиве `MAPS` внутри `index.html`. Чтобы добавить карту — положи её изображение рядом и добавь строку в `MAPS`.
