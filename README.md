# 🧭 StarWarsRP — навигатор по картам

Веб-приложение для ролевых игр по Star Wars: кликаешь по карте — получаешь **координаты клетки**, **расстояние** и **азимут** между точками, а для клона/техники — ещё и **время в пути**. Работает прямо в браузере, ничего устанавливать не нужно, интернет нужен только чтобы открыть страницу (дальше — офлайн).

Есть **две версии**:

| | Онлайн-сайт (сейчас доступен всем) | Полная версия (пока только локально) |
|---|---|---|
| Карт | 22 (Phase-2) | 22 (Phase-2) |
| Переключатель Phase-1/Phase-2 | ✅ (Phase-1 пока пуст) | ✅ (Phase-1 пока пуст) |
| Расстояние + азимут | ✅ | ✅ |
| Типы «Клон / Транспорт / Оружие» | ❌ нет | ✅ |
| Маршрут из нескольких точек | ❌ нет (только 2 точки) | ✅ (для клона/техники) |
| Время в пути | ❌ нет | ✅ (частично откалибровано) |
| Где взять | просто открыть ссылку | скачать `index.html` + карты из этой папки |

Дальше — подробно про обе версии.

## 🌐 Версия 1: онлайн-сайт (доступна прямо сейчас)

**https://lereactor.github.io/starwars-navigator/**

Просто перейди по ссылке в браузере (телефон/компьютер — не важно). Ссылку можно скидывать кому угодно, регистрация не нужна.

Репозиторий: https://github.com/Lereactor/starwars-navigator

### Что умеет

- **Карта**: выпадающий список из 22 карт (Phase-2) — МЦБ-17/9/3-18, Татуин, Татуин-город, Альдераан, Анаксес, Фелуция, Джеонозис (улитка/база/завод/гл. улей), Корусант-подземный, Луна Риши, Мигетто, Мигетто-2, Морабанд, Набу (ангар/лес), Рен Вар, Рен Вар-горы, Рилот, Такодана, Утапау.
- **Фаза**: кнопки Phase-1 / Phase-2 над картой. Сейчас все карты — Phase-2, поэтому при нажатии Phase-1 список карт будет пустым («— карт пока нет —») — это ожидаемо, карты для первой фазы ещё не добавлены.
- **Расчёт**: клик по карте — 1-я точка («Я»), второй клик — 2-я точка («Цель»). Между ними сразу считаются:
  - координаты клеток обеих точек (например, `H8`);
  - расстояние — в игровых единицах и в клетках;
  - азимут от 0° до 360° (0° = север, по часовой стрелке: восток = 90°, юг = 180°, запад = 270°);
  - линия между точками прямо на карте.
- **Третий клик** — сбрасывает обе точки и ставит новую первую точку заново.
- **Точки можно перетаскивать** мышью/пальцем, чтобы поправить позицию не пересчитывая всё заново.
- **Сторона клетки** — поле ввода (по умолчанию 135 ед.), меняется вручную под конкретную карту, если размер клетки другой.
- **Направление севера** — кнопка «🧭 Север», проворачивает систему отсчёта на 90° за клик (если на карте север направлен не «вверх», а в другую сторону).
- **Зум**: колесо мыши, кнопки `+` / `−` / `⟲` (сброс) в углу, или щипок двумя пальцами на телефоне.
- **Панорама** (сдвиг карты): зажатая правая кнопка мыши, средняя кнопка, Ctrl+перетаскивание — или одним пальцем при обычном зуме на телефоне.

### Чего пока нет на сайте

Разработана, но ещё не опубликована на сайте расширенная версия с типами перемещения (клон/транспорт/оружие), маршрутом из нескольких точек и расчётом времени в пути — см. раздел ниже. Пока это можно запустить только локально.

## 🚀 Версия 2: полная (клон / транспорт / оружие, маршруты, время) — пока локально

Эта версия лежит в этой папке (`index.html` + все карты) и в подпапке `publish\` — на сайте её ещё нет, скачивается и запускается вручную (см. «Запуск локально» ниже).

### Что добавлено сверх онлайн-версии

**Переключатель «Тип»** над картой — три режима работы:

1. **🚶 Клон** — маршрут из любого числа точек подряд. Режимы: *Пешком*, *Лёгкий бег*, *Быстрый бег*. Каждый клик по карте добавляет новую точку маршрута (не только 2, а сколько нужно). Показывается таблица участков (расстояние + время + азимут каждого отрезка) и итоговая строка по всему маршруту.
2. **🚁 Транспорт** — тот же маршрут из точек, но режимы: *LAAT/i*, *ARC-170*, *TX-130*, *AT-RT*. Дополнительно выбирается **скорость** (100 / 300 / 1000) — время пересчитывается математически под неё.
3. **💣 Оружие** — старое поведение как на сайте: 2 точки «Я → Цель», только расстояние и азимут (без времени), 3-й клик сбрасывает пару.

**Калибровка времени** — под переключателем режима появляется поле «Калибровка» (секунд на клетку). Его можно менять прямо в интерфейсе, без правки кода — как и поле «Сторона клетки». Пока для режима не введено число, время показывается как «—».

Текущее состояние калибровки:

| Тип | Режим | Калибровано? | Значение |
|---|---|---|---|
| Клон | Пешком | ✅ | 100 с/клетку |
| Клон | Лёгкий бег | ✅ | 35 с/клетку |
| Клон | Быстрый бег | ✅ | 22 с/клетку |
| Транспорт | LAAT/i | ✅ | 18.53 с/клетку при скорости 100 |
| Транспорт | ARC-170 | ❌ нет | — |
| Транспорт | TX-130 | ❌ нет | — |
| Транспорт | AT-RT | ❌ нет | — |

Значения для ARC-170, TX-130 и AT-RT ещё предстоит замерить — как только они появятся, их можно будет вписать прямо в поле «Калибровка» в интерфейсе (или сразу в код, см. «Обновление» ниже).

### Как пользоваться маршрутом (Клон / Транспорт)

1. Выбери тип «Клон» или «Транспорт» и нужный режим/скорость.
2. Кликай по карте — каждая точка добавляется в конец маршрута (первая — голубая, последняя — жёлтая, промежуточные — голубовато-серые).
3. Внизу появляется таблица: каждый участок пути (1→2, 2→3, …) с расстоянием, временем и азимутом, и строка «ИТОГО» с суммой по всему маршруту.
4. «↶ Убрать последнюю» — удаляет последнюю добавленную точку (если поставил лишнюю).
5. «✕ Сброс» — очищает весь маршрут.
6. Точки можно перетаскивать мышью — таблица пересчитается автоматически.

Для «Оружие» логика прежняя: 2 точки, 3-й клик — новая пара.

Остальное (карта, фаза, зум, панорама, сторона клетки, направление севера) работает так же, как в онлайн-версии.

## 💻 Запуск локально

Нужны `index.html` и все файлы карт (`*.jpg`, `*.png`) — вместе, в одной папке.

### Вариант 1 — через Git

```bash
git clone https://github.com/Lereactor/starwars-navigator.git
cd starwars-navigator
```

Это даст **онлайн-версию** (без типов клон/транспорт/оружие) — репозиторий на GitHub пока не обновлён до полной версии.

Чтобы получить **полную версию** — скопируй `index.html` и все карты из этой папки (`C:\Claude\StarWarsRP-map\`) или из подпапки `publish\`.

### Вариант 2 — скачать архивом (без Git)

- Готовый архив с полной версией уже собран: **`StarWars-Navigator.zip`** в этой папке. Распакуй его в отдельную папку и открой `index.html`.
- Или с GitHub (онлайн-версия): открой https://github.com/Lereactor/starwars-navigator → зелёная кнопка **Code** → **Download ZIP** → распакуй → открой `index.html`.

> Важно: не открывать `index.html` прямо из архива — сначала распаковать. Все файлы карт должны лежать рядом с `index.html` в одной папке, иначе картинки не подтянутся. Интернет после скачивания не нужен — всё работает офлайн.

## 🔄 Обновление / добавление карт и режимов

- Карты и их параметры (размер сетки, направление строк, сторона клетки, фаза) описаны в массиве `MAPS` внутри `index.html`. Чтобы добавить карту — положи изображение рядом и добавь строку в `MAPS` (в т.ч. можно добавить карты с `phase: 1`, тогда кнопка «Phase-1» перестанет быть пустой).
- Типы расчёта и режимы (клон/транспорт/оружие) описаны в объекте `CATEGORIES` внутри `index.html`. Калибровочные значения (`secPerCell` / `secPerCell100`) заданы там по умолчанию — их можно поменять прямо в открытом приложении полем «Калибровка» (без правки кода), но при перезагрузке страницы вернутся значения по умолчанию из кода, так что итоговые замеры лучше сразу вписывать и в код.

## 📦 Публикация (для справки)
